# Welcome Home Whistler

This project is a web application for managing vacation rentals in Whistler. It allows users to browse listings, book accommodations, and manage their reservations.

## Features
- User authentication
- Listing management
- Booking system

## Installation
To install the project, clone the repository and run the following commands:

```bash
npm install
npm run dev
```

## Usage
After installation, you can start the development server with:

```bash
npm run dev
```
